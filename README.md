# Hi, I'm Shivani Bhat 👋

Final-year Electronics & Telecommunications student at SPIT Mumbai, specialising in embedded systems, FPGA design, and GPU computing. I've deployed real hardware at TIFR (live cosmic muon detector on Spartan 7 + Cyclone IV FPGAs), written HLS designs at NIELIT using Vitis HLS and Vivado, and built systems that run in the real world, not just on breadboards.
